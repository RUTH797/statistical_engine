Project Overview
Pure-Python statistical engine from scratch demonstrating Law of Large Numbers.

## Mathematical Logic
- Population Variance: σ² = Σ(x - μ)² / N
- Sample Variance: s² = Σ(x - x̄)² / (n - 1) [Bessel's correction]
